# Story 1.2: Real-time UX Logs

## Contexto
Os usuários ficam "no escuro" durante a execução dos agentes (3-6 minutos), vendo apenas uma barra de progresso.

## Descrição
Como Usuário, quero ver o que cada agente está fazendo em tempo real para ter confiança no progresso do sistema.

## Critérios de Aceite
- [ ] Implementar `st.status()` no loop de execução.
- [ ] Exibir logs parciais (outputs) dos agentes conforme eles terminam suas tasks.

## Estimativa
6 horas.

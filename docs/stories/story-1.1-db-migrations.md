# Story 1.1: Database Migrations

## Contexto
Atualmente, o banco de dados SQLite é gerenciado por comandos manuais no código, dificultando a evolução e manutenção.

## Descrição
Como Desenvolvedor, quero implementar um framework de migração (Alembic) para versionar o esquema do banco de dados de forma segura.

## Critérios de Aceite
- [ ] Alembic instalado e configurado no projeto.
- [ ] Primeiro snapshot do schema atual (`demandas`, `resultados`) gerado.
- [ ] Documentação de como rodar novas migrações.

## Estimativa
4 horas.
